# Taxi Dispatcher

Браузерный симулятор таксопарка на карте реального города: принимайте заказы, следите за машинами и водителями, покупайте гаражи и офисы, развивайте бизнес.

## Демо

**https://etunimisukunimi.github.io/taxi-dispatcher/**

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте адрес из терминала (обычно `http://localhost:5173`).

## Стек

- React 19, TypeScript, Vite
- Leaflet / react-leaflet
- Zustand
- OSRM (маршруты), CARTO/OSM (тайлы карты)
